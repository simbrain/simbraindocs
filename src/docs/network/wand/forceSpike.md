# Force Spike Action

Forces touched spiking neurons to spike by setting `isSpike` to true. Non-spiking neurons are ignored.

# Parameters

This action has no action-specific parameters.

For [common properties](.#common-properties) like Letter, Color, and Radius, see the main wand tool page.

# Using Force Spike

1. Select the Force Spike action in the wand palette
2. Activate wand mode if not already active
3. Paint over spiking neurons to force spikes
4. Use the raster plot or spike highlighting to observe the spike propagation through the network
